4. Using Filters

4.1 Global Filters

Located at [top/right side], these filters affect all visuals:

  • Date Range: Select start and end dates
  • Region: Select one or multiple regions
  • [Filter Name]: [Description]

4.2 Visual-Level Filters

Click on any visual element to filter other visuals on the page.

4.3 Clearing Filters

  • Click the eraser icon in the filter pane
  • Or press Ctrl+Click on the selected filter

5. Exporting Data

5.1 Export to Excel

  1. Click on the visual
  2. Click the ... (more options) menu
  3. Select Export data
  4. Choose Summarized data or Underlying data

5.2 Export to PDF

  1. Click File > Export to PDF
  2. Select pages to include
  3. Click Export

5.3 Export to PowerPoint

  1. Click File > Export to PowerPoint
  2. Choose current values or default values
  3. Click Export
